# NVB Python Capstone -- Weekly Personal Study Tracker
NVB Python Summer 2026 Capstone Project.

## What it does
My program tracks the subjects and the time you spent studying it for the past week. It has a beginner-friendly UI and it's helpful with keeping track of your study time so you can reflect on what you've done and potentially make changes to your study routine. For each day, the program tracks, in a table, the subject you studied, how many hours you spent studying it, and summarizes everything in a concise report. The report shows the total number of hours you spent studying, the number of hours spent studying each subject and which subject you spent the most time on.

## How to run it
1. Open the notebook in Google Colab
2. Run all cells from top to bottom
3. Link to CoLab Notebook --> https://colab.research.google.com/drive/1IGzPFPGgy3NA5EH81SHU414EPssUTpLB?usp=sharing

- Pandas DataFrames
- Pandas Series
- "For" Loops
- Functions

## What it produces
The final result is a summary of the subjects and hours studied for each day of the past week.
